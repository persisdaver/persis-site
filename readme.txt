# Persis Daver Personal Website

## How to Publish

1. Create a free GitHub account at https://github.com/
2. Create a new public repository (e.g., persis-daver-site)
3. Upload all HTML files and the images folder
4. Go to Settings > Pages, select main branch and /root
5. Your site will be live at https://yourusername.github.io/persis-daver-site/

## Contact Form

- Register at https://formspree.io/ with your email (persisdaver@hotmail.com)
- Replace 'yourformid' in contact.html with your Formspree form ID

## Customization

- Add or replace images in the images folder
- Edit testimonials, blog posts, and articles in the HTML files as needed
